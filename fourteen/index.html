Suggested path; - * setup a sketch and run it through our localhost, please download and unzip 'p5-fresh-example-netart.zip'

We will need to open the 'Terminal', and you will read something like fulanito:~ myname$ in front of this line you will type cd and you will drag & drop the folder that came out when you unzipped the previous .zip, so it will look like the following line: fulanito:~ myname$ cd /Users/myname/Documents/eastern_bloc_atelier/p5-fresh-example-netart now you will need to press enter ↵

Now we are inside of the specific folder that contains our project, so we are going to read in our terminal that address/folder: fulanito:p5-fresh-example-netart myname$ and we will write in front the following line: python -m SimpleHTTPServer 8080

The following step is to open our browser and type in the URL bar localhost:8080, it should automatically open our sketch-example.

Now you're ready to open the contents of the folder in Sublime Text, or your favoirte code/text editor; in Sublime I recommend you to split the window in 3 parts by pressing 'cmd+alt+3', so now, you can open index.html, sketch.js and style.css, those 3 files are the main ones that you will be editing, all the p5.js related stuff is controlled mainly through the sketch.js, so now you can make changes to the sketch and save cmd+s, after you will need to click into the website where your sketch is running and refresh cmd+r, et voila!, you'll see the changes <3

* Advise: ~ Create a folder for this workshop or for p5.js explorations / Create or save a copy of the downloaded repository / So you can start new projects from this basis / Keep your files in order (:
